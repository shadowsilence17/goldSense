# Gold Price Prediction Web App

Flask application for predicting gold prices using machine learning.

## Quick Start

```bash
# Make sure models are trained first (run from project root)
cd ..
python3 train_model_for_webapp.py

# Return to webapp and install dependencies
cd webapp
pip install -r requirements.txt

# Run the app
python3 app.py
```

Open: http://localhost:5000

## API Endpoints

### Health Check
```bash
GET /health
```

### Predict Gold Price
```bash
POST /api/predict
Content-Type: application/json

{
  "type": "day"    # or "week" or "month"
}
```

## Response Format

```json
{
  "success": true,
  "timestamp": "2024-10-26T08:24:00.000Z",
  "current_price": 2650.50,
  "prediction": {
    "next_day": 2655.30,
    "change": 4.80,
    "change_percent": 0.18
  }
}
```

## Required Model Files

The app needs these files in `models/` directory:
- `best_model.pkl` - Trained ML model
- `scaler_X.pkl` - Feature scaler
- `scaler_y.pkl` - Target scaler
- `feature_names.pkl` - Feature names

Generated by running `train_model_for_webapp.py` from project root.

## Production Deployment

```bash
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

## Docker

```bash
# From project root
docker build -t gold-predictor .

```
